12. März 2026 Open Source

Qwen 3.5 Release

Mit dem Release von Qwen 3.5 wurden auch diverse Destillationen veröffentlicht. Für uns besonders spannend: die 122B-Version. Sie ist ein direkter Konkurrent zu GPT-OSS-120B, läuft auf derselben Hardware, und übertrifft das Vorgängermodell in praktisch allen Benchmarks.

Was die neue Generation mitbringt:

→ Vision: Das Modell versteht Bilder und Dokumente nativ. Rechnungen parsen, Pläne lesen, Screenshots analysieren.

→ Stärkeres Reasoning: Bei Allgemeinwissen, Programmieren und Tool-Nutzung schlägt es sogar GPT-5 mini, ein proprietäres Modell von OpenAI.

→ Kein Hardware-Upgrade nötig: Wer heute GPT-OSS-120B betreibt, kann direkt wechseln.

Für Schweizer Unternehmen heisst das: Frontier-nahe KI-Leistung, inklusive Dokumentenverarbeitung, auf eigener Infrastruktur. Ohne API-Abhängigkeit, ohne Datenabfluss.

Die Open-Source-Welt schliesst die Lücke schneller, als viele erwarten.
